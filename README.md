# Getolog Manager Web Admin Panel

Профессиональная веб-админ-панель для управления системой Getolog с чистым и простым дизайном.

## Технологии

- **React 18** - UI фреймворк
- **Vite** - Быстрый build tool
- **React Router** - Маршрутизация
- **Tailwind CSS** - Утилитарный CSS фреймворк
- **Axios** - HTTP клиент
- **React Icons** - Иконки
- **Recharts** - Графики и диаграммы

## Установка

1. Перейдите в папку frontend:
```bash
cd frontend
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте `.env` файл:
```env
VITE_API_URL=http://localhost:5000/api
VITE_ADMIN_ID=7547827275
```

4. Запустите development сервер:
```bash
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Функции

### Dashboard (Главная страница)
- Глобальная статистика системы
- Карточки с ключевыми метриками
- Детальная разбивка по категориям
- Последние активности (клиенты, транзакции)

### Clients (Клиенты)
- Таблица всех клиентов с пагинацией
- Поиск по ID или username
- Просмотр баланса, плана, количества ботов и пользователей
- Управление балансом (добавить/вычесть)
- Модальное окно для изменения баланса

### Bots (Боты)
- Список всех ботов с информацией о владельце
- Статус ботов (активен/неактивен)
- Количество пользователей каждого бота
- Пагинация для больших списков

### Users (Пользователи)
- Таблица всех пользователей системы
- Фильтрация по статусу (активен/удален/free)
- Информация о подписке и балансе
- Связь с владельцем бота

### Transactions (Транзакции)
- История всех транзакций
- Фильтрация по статусу (одобрено/ожидает/отклонено)
- Информация о сумме, типе операции
- Связь с пользователем и клиентом

### Export (Экспорт)
- Экспорт всех данных в Excel
- Экспорт только клиентов
- Экспорт только транзакций
- Автоматическая загрузка файла

## Компоненты

### StatCard
Карточка статистики с иконкой, заголовком и значением:
```jsx
<StatCard
  icon={FaUsers}
  title="Jami Clientlar"
  value={150}
  color="blue"
/>
```

### Loading
Индикатор загрузки:
```jsx
<Loading size="large" />
```

### EmptyState
Пустое состояние с иконкой:
```jsx
<EmptyState message="Ma'lumot topilmadi" icon={FaInbox} />
```

## API Integration

Все API запросы обрабатываются через `src/services/api.js` с использованием Axios.

Примеры:
```javascript
// Получить статистику
const stats = await getGlobalStats();

// Получить клиентов
const clients = await getClients(page, limit);

// Поиск клиента
const results = await searchClients('username');

// Обновить баланс
await updateClientBalance(userId, 50000, 'add');
```

## Дополнительная настройка

### Изменить порт dev сервера
В `vite.config.js`:
```javascript
server: {
  port: 3000, // изменить порт
}
```

### Изменить API URL
В `.env`:
```env
VITE_API_URL=http://your-api-url.com/api
```

## Troubleshooting

**Ошибка при установке зависимостей:**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Backend API недоступен:**
- Убедитесь, что backend запущен на `http://localhost:5000`
- Проверьте настройки CORS в backend
- Проверьте `VITE_API_URL` в `.env`
